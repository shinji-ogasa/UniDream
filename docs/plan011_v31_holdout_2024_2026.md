# Plan011 v31 2024-2026 Untouched Holdout

このメモは、Plan011 v31 の完全未使用期間に対する保存済みAC checkpoint replay結果。VC/外部説明では、fold0-12 の開発WFOとは分けて扱う。

## Scope

- Config: `configs/plan011_overlay_actor_v31_holdout.yaml`
- Replay checkpoint dir: `checkpoints/plan011_overlay_actor_v31_relative_constraint_ac_s007`
- Symbol / interval: `BTCUSDT`, `15m`
- Data range: `2018-01-01` to `2026-04-17`
- Holdout folds: `15-23`
- Holdout test span: `2024-01-16` to `2026-04-16`
- Seed/device for replay: `7` / `cpu`
- Costs: default profile, one-way full `Delta position = 1` cost `5.50bps`

AlphaEx は strategy の最終total return minus B&Hの最終total return。年率換算ではない。MaxDDDelta は strategy の絶対MaxDD minus B&Hの絶対MaxDDで、マイナスが改善。

Each fold keeps the same WFO structure:

- train: 2 years
- validation: 3 months
- test: 3 months
- slide: 3 months

## Interpretation

The untouched 2024+ holdout is weak under the current final-value AlphaEx definition.

What survived:

- Low turnover survived: mean turnover `0.60`, max `1.73`.
- The model stayed near B&H instead of collapsing into high-turnover trading.
- The runtime uses actual WM -> BC -> AC model inference, not a legacy rule overlay.

What did not survive:

- The model did not improve drawdown: MaxDDDelta was positive in every holdout fold.
- AlphaEx was only near-neutral: mean `+0.11pt`, median `-0.04pt`.
- No fold met `AlphaEx >= +3pt`.
- Sharpe improvement did not appear in aggregate.

Investor-safe claim:

> Plan011 v31 is a low-turnover B&H-relative neural overlay research artifact. It is not yet a drawdown-reduction model or a production-grade alpha model.

Do not claim `AlphaEx >= +3pt` or `MaxDDDelta <= -3pt` on untouched holdout. The current evidence supports continued research, not production-grade risk improvement.
